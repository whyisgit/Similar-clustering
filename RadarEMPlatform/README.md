## 多体制雷达电磁信号平台

仅生成脉冲参数元数据（不生成数字信号），满足每秒约百万脉冲的规模，并提供两界面：

- 界面一：生成各种类型信号（横轴到达时间，纵轴PRI），参数可设置或采用默认值。
- 界面二：选择混合信号，显示三个图：
  - 到达时间-频率
  - 到达时间-脉宽
  - 到达时间-脉幅

- 运行脚本：
   ```matlab
   run('RadarEMPlatform/RunRadar.m')
   ```

## 已支持的脉间调制类型（>10类）

- 重频固定：PRI恒定
- 重频抖动：围绕基础PRI随机抖动（可设置抖动比例）
- 重频参差：按模式序列循环（如 `[0.9,1.0,1.1,1.0]`）
- 单线性滑变：PRI线性上升（或下降）
- 双线性滑变：两段线性（上升后下降）
- 正弦滑变：PRI按正弦规律变化（可设置周期数与振幅比例）
- 重频组变：按组改变PRI（每组恒定，组间步进）
- 频率组变：按组改变载频（频率集合循环使用）
- 频率捷变：每脉冲载频在给定集合中随机切换
- 排定信号：按排定序列生成PRI（可设置相对系数序列）

> 说明：除涉及频率变化的类型（频率组变/频率捷变）外，载频默认恒定；脉宽、脉幅、到达角均可设定为常量。到达时间 `TOA` 由 `PRI` 的累计和得到（首个从0开始）。

## 参数说明

- 时长 `duration(s)`：总模拟时长。默认 1s。
- 基础PRI `basePRI(s)`：默认 1e-6 s（1 微秒），可达约百万脉冲/秒。
- 脉宽 `pulseWidth(s)`、脉幅 `amplitude`、载频 `fc(Hz)`、到达角 `aoa(°)`：默认值可修改。
- 抖动比例 `jitterSpanRatio`：抖动幅度相对基础PRI的比例，默认 0.2。
- 组大小 `groupSize`：组变的每组脉冲数量，默认 100。
- 其他高级参数（参差模式、线性幅度、正弦振幅与周期、频率集合、排定序列）可在代码内 `SignalGenerator.generate(cfg)` 的配置中扩展。

## 绘图与性能

- 绘图对超过 `200000` 点自动降采样，保证交互响应。
- 生成过程完全向量化，满足百万脉冲/秒的元数据生成需求。

## 目录结构

- `SignalGenerator.m`：信号生成器（核心），根据类型生成 `toa/pri/width/amp/freq/aoa`。
- `RadarApp.m`：双界面应用（界面一生成，界面二混合）。
- `RunRadar.m`：启动脚本。
